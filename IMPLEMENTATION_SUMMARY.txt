TECHNICAL ARCHITECTURE
================================================================================

STRUCTURE:
lib/
├── core/                  # Core utilities and configuration
│   ├── constants/         # API endpoints, app constants
│   ├── network/          # Dio client, interceptors, API response
│   ├── router/           # GoRouter configuration with auth guards
│   ├── storage/          # Secure token storage service
│   ├── theme/            # Custom Material Design 3 theme
│   └── utils/            # Helpers, validators, date utilities
├── data/                  # Data layer
│   ├── models/           # Data models with JSON serialization
│   ├── repositories/     # Repository pattern implementation
│   └── services/         # API services
├── presentation/          # Presentation layer
│   ├── providers/        # Riverpod state management
│   ├── screens/          # All UI screens
│   └── widgets/          # Reusable UI components
└── main.dart             # App entry point

STATE MANAGEMENT (Riverpod):
• AuthProvider - Authentication state & actions
• TasksProvider - Tasks list, filtering, updates
• NotificationsProvider - Notifications & unread count

NAVIGATION (GoRouter):
• /login - Login screen
• /dashboard - Main dashboard (default after login)
• /tasks - Tasks list
• /tasks/:id - Task details
• /tasks/:id/report - Create task report
• /notifications - Notifications list
• /products - Products list
• /products/:id - Product details
• /suppliers - Suppliers list
• /suppliers/:id - Supplier details
• /profile - User profile

NETWORK LAYER:
• Dio HTTP client with interceptors
• AuthInterceptor for JWT injection
• Automatic token refresh on 401
• Request retry after refresh
• Comprehensive error handling
• Logging in debug mode

THEME:
• Primary: Orange (#FF8C42)
• Background: Cream/Beige (#FFF8F0, #FFFBF5)
• Success: Green (#00B894)
• Error: Red (#D63031)
• Warning: Yellow (#FDCB6E)
• Rounded cards (16px), buttons (12px)
• Material Design 3

HOW TO RUN
================================================================================

1. SETUP:
   flutter pub get

2. CONFIGURE:
   Edit lib/core/constants/app_constants.dart:
   static const String baseUrl = 'https://your-api-url.com';

3. RUN:
   flutter run

4. BUILD (Production):
   Android: flutter build apk --release
   iOS: flutter build ios --release
